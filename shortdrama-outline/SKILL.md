# shortdrama-outline - 短剧大纲与付费卡点规划技能

## 功能描述
规划短剧的整体结构，包括 100 集分集大纲、付费卡点设计、节奏控制。

## 100 集标准结构

```
【第一阶段】1-30 集：建置 + 第一个大高潮
├─ 1-10 集：免费期，建立期待
├─ 第 10 集：第一付费卡点
└─ 11-30 集：持续爽感，关系发展

【第二阶段】31-60 集：升级 + 第二个大高潮
├─ 第 30 集：第二付费卡点
└─ 31-60 集：矛盾升级，反派施压

【第三阶段】61-100 集：决战 + 结局
├─ 第 60 集：第三付费卡点
├─ 61-90 集：最终对决
└─ 91-100 集：收尾 + 圆满
```

## 付费卡点设计

### 第一卡点（第 10 集）
- **目的**：转化免费用户为付费用户
- **设计**：第一个大高潮 + 强悬念
- **示例**：女主身份即将揭晓，戛然而止

### 第二卡点（第 30 集）
- **目的**：防止用户中途弃剧
- **设计**：关系突破/重大反转
- **示例**：男女主终于表白，但出现第三者

### 第三卡点（第 60 集）
- **目的**：冲刺大结局
- **设计**：终极对决前的最大危机
- **示例**：主角陷入绝境，看似必败

- [ ] 节奏控制合理吗？
