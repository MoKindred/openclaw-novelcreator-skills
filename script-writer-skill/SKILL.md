# Script Writer Skill - 专业短剧编剧技能

## 短剧核心要素

### 短剧特点

| 要素 | 要求 |
|------|------|
| **时长** | 1-3 分钟/集（约 300-500 字/集） |
| **集数** | 80-100 集 |
| **节奏** | 极快，每集至少 1 个情节点 |
| **冲突** | 强冲突，情绪激烈 |
| **爽点** | 密集，打脸/逆袭/复仇 |
| **悬念** | 每集结尾必须留悬念 |

### 热门题材

- 战神/龙王归来
- 豪门/总裁/千金
- 逆袭/打脸/复仇
- 甜宠/先婚后爱
- 萌宝/亲子
- 玄幻/修仙

## 剧本结构

### 三幕式结构（单集）

```
开场（15 秒）→ 建立场景和人物
发展（45 秒）→ 冲突升级
高潮（30 秒）→ 反转/打脸
结尾（15 秒）→ 悬念钩子
```

### 整体结构（80 集）

```
第 1-10 集：铺垫期（身份隐藏，被欺负）
第 11-30 集：发展期（小试牛刀，局部打脸）
第 31-60 集：高潮期（身份曝光，大规模打脸）
第 61-80 集：收尾期（最终对决，圆满结局）
```

## 冲突设计

### 冲突类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **身份冲突** | 隐藏身份 vs 被轻视 | 总裁装穷被羞辱 |
| **情感冲突** | 爱恨情仇 | 前男友/前女友 |
| **利益冲突** | 财产/权力争夺 | 遗产/公司控制权 |
| **尊严冲突** | 被侮辱后反击 | 当众打脸 |

### 冲突升级公式

```
小冲突 → 中等冲突 → 大冲突 → 终极冲突
（被嘲讽）（被欺负）（被陷害）（生死对决）
```

## 台词编写

### 台词特点

- **短促有力**：每句不超过 20 字
- **情绪强烈**：愤怒、震惊、不屑
- **口语化**：符合角色身份
- **有记忆点**：金句/口头禅

### 经典台词模板

```
"你算什么东西，也配跟我说话？"
"三年之期已到，恭迎龙王归来！"
"我动你，又如何？"
"有眼不识泰山！"
"今天，我要让你们都付出代价！"
```

### 标准格式

```markdown

**[角色 B]**：（情绪）台词

**动作**：[动作描述]

**【本集悬念】**
[结尾钩子]
```

## 创作流程

### 步骤 1：接收灵感

记录用户提供的：
- 核心创意
- 主角设定
- 期望题材
- 特殊要求

### 步骤 2：故事大纲

输出：
- 一句话故事
- 主要人物
- 核心冲突
- 分集大纲（前 10 集详细）

### 步骤 3：人物设定

为每个主要角色创建档案

### 步骤 4：编写剧本

按集编写，每集包含：
- 场景描述
- 对话
- 动作指示
- 结尾悬念

### 步骤 5：审核优化

检查：
- [ ] 节奏是否够快
- [ ] 冲突是否够强
- [ ] 爽点是否密集
- [ ] 悬念是否到位
- [ ] 台词是否口语化

## 注意事项

1. **节奏第一**：宁可牺牲逻辑，不能牺牲节奏
2. **情绪至上**：让观众爽比什么都重要
3. **悬念不断**：每集必须留钩子
4. **台词精炼**：能说短句不说长句
5. **冲突密集**：没有冲突就制造冲突
