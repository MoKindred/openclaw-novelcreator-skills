# Production Design Skill - 美术设计技能

## 核心理念

**专业短剧拍摄蓝本需要完整的美术设计方案。**

- **角色设计**：人物造型、服装、化妆、发型
- **场景设计**：空间布局、色调、光影、氛围
- **道具设计**：关键道具、陈设道具、消耗道具

---

## 职位分工

### 角色设计师（Character Designer）

**职责**：
- 人物造型设计（服装/化妆/发型）
- 角色气质把控（年龄/身份/性格外化）
- 多套服装变体设计
- 特殊妆容设计（伤痕/老年妆等）

**输出**：`角色设计表.md`

### 场景设计师（Set Designer）

**职责**：
- 空间布局设计（平面图/立面图）
- 色调和氛围设计
- 光影方案设计
- 陈设道具布置

**输出**：`场景设计表.md`

### 道具设计师（Prop Designer）

**职责**：
- 关键道具设计（推动剧情的道具）
- 陈设道具清单（场景布置）
- 消耗道具清单（拍摄消耗）
- 道具来源方案（购买/租赁/定制）

**输出**：`道具设计表.md`

## 创作流程

### 步骤 1：读取剧本和导演讲戏

读取：
- `01-short-drama-script.md`（剧本）
- `03-director-notes.md`（导演讲戏）

提取：
- 所有角色及其造型需求
- 所有场景及其氛围需求
- 所有道具及其功能需求

### 步骤 2：角色设计

为每个主要角色设计：
1. 发型设计
2. 妆容设计
3. 服装变体（2-3 套）
4. 配饰细节
5. 气质要点

### 步骤 3：场景设计

为每个主要场景设计：
1. 空间布局（平面/立面）
2. 色调方案
3. 光影方案
4. 陈设布置

### 步骤 4：道具设计

设计所有道具：
1. 关键道具（详细设计）
2. 陈设道具（按场景）
3. 消耗道具（按集数）
4. 预算估算

### 步骤 5：审核

检查：
- [ ] 角色造型符合身份和性格
- [ ] 场景色调符合氛围
- [ ] 道具设计符合剧情需求
- [ ] 预算在可控范围内

### 与服化道参考图的关系

| 职位 | 输出 | 用途 |
|------|------|------|
| **服化道** | 参考图提示词 | AI 生成参考图 |
| **美术设计** | 详细设计表 | 实际拍摄/制作参考 |

**工作流程**：
```
剧本 → 服化道（AI 参考图） → 美术设计（详细设计表） → 实际制作
```

---

## 注意事项

1. **角色设计**：考虑演员实际条件，设计可执行的造型
2. **场景设计**：考虑实际拍摄场地限制
3. **道具设计**：考虑预算和来源可行性
4. **一致性**：角色/场景/道具之间风格统一
5. **可执行性**：设计方案要能实际落地

---

*美术设计的任务：为短剧提供完整的视觉设计方案，确保拍摄可执行*
