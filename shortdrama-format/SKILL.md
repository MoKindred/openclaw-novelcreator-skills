# shortdrama-format - 短剧剧本格式规范化技能

## 功能描述
将任意文本内容转换为标准短剧剧本格式，确保符合行业规范。

## 输入
- 故事梗概/小说片段/创意想法
- 可选：指定集数、场景、人物

## 输出
- 标准格式的短剧剧本（Markdown 格式）
- 包含：场次信息、场景、时间、人物、镜头、动作、台词

## 格式规范

### 基础结构
```markdown
【第 X 集 第 X 场】

场景：室内/室外 + 具体地点
时间：日戏/夜戏/晨/昏
人物：角色 A、角色 B
镜头：远景/全景/中景/近景/特写

---

[动作描述/环境描写]

角色 A：
（情绪/动作提示）
台词内容...

角色 B：
台词内容...

【本集完】
（悬念钩子）
```

### 关键规则
1. **单集字数**：500-800 字
2. **台词长度**：不超过 2 行
3. **角色数量**：单集有台词的角色≤20 个
4. **时长控制**：每集 1-2 分钟
5. **结尾钩子**：必须留悬念

## 使用示例

### 输入
```
帮我写第 1 集，女主在婚礼上被渣男抛弃，遇到男主。
```

### 输出
```markdown
【第 1 集 第 1 场】

场景：室内 - 婚礼现场
时间：日戏
人物：女主（林浅）、男主（顾墨）、新郎、新娘、宾客
镜头：全景→中景→特写

---

【全景】豪华婚礼现场，宾客满座

【中景】新娘挽着父亲走向新郎

【特写】新娘手机震动，脸色骤变

新娘：
（接电话）
你说什么？他在医院？

【近景】新郎焦急

新郎：
怎么了？

新娘：
（挂电话）
我们分手吧。

【全景】全场哗然，新娘跑向门口

【中景】女主与新娘相撞，文件散落

女主：
（摔倒）
啊...

【特写】男主走近，伸出手

男主：
你没事吧？

女主：
（抬头，愣住）
我...没事。

【近景】男主盯着女主脸上的胎记

男主：
（眼神复杂）
我们是不是...见过？

【悬念】女主慌乱遮住胎记，画面定格

【本集完】
```

## 注意事项
1. 动作描述用 [] 包裹
2. 情绪/动作提示用（）包裹
3. 镜头语言用【】标注
4. 每场戏另起一段
5. 台词简洁有力，避免水词

## 质量检查
- [ ] 场次信息完整
- [ ] 场景/时间/人物明确
- [ ] 镜头语言清晰
- [ ] 台词不超过 2 行
- [ ] 有动作描述
- [ ] 结尾有钩子
- [ ] 字数 500-800 字
