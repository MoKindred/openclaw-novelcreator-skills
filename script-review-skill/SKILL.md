# Script Review Skill - 剧本审核清单

## 审核维度

### 节奏审核

| 检查项 | 标准 | 问题标注 |
|--------|------|----------|
| 开场速度 | 15 秒内进入冲突 | |
| 情节点密度 | 每集至少 1 个情节点 | |
| 对话节奏 | 每句不超过 20 字 | |
| 场景转换 | 快速切换，不拖沓 | |

### 冲突审核

| 检查项 | 标准 | 问题标注 |
|--------|------|----------|
| 冲突强度 | 情绪激烈，不温吞 | |
| 冲突升级 | 每集比上集更激烈 | |
| 反派智商 | 足够坏，但不过于弱智 | |
| 打脸力度 | 够狠，够爽 | |

### 爽点审核

| 检查项 | 标准 | 问题标注 |
|--------|------|----------|
| 爽点密度 | 每集至少 1 个爽点 | |
| 爽点类型 | 多样化（打脸/逆袭/曝光）| |
| 爽点时机 | 在中后段，铺垫充分 | |

### 悬念审核

| 检查项 | 标准 | 问题标注 |
|--------|------|----------|
| 结尾钩子 | 每集必须留悬念 | |
| 悬念类型 | 身份/危险/情感/秘密 | |
| 悬念力度 | 足够吸引看下集 | |

### 台词审核

| 检查项 | 标准 | 问题标注 |
|--------|------|----------|
| 口语化 | 符合角色身份 | |
| 记忆点 | 有金句/口头禅 | |
| 情绪化 | 情绪强烈，不平淡 | |

## 审核报告格式

```markdown
# 剧本审核报告

## 基本信息
- 剧本名称：
- 集数范围：
- 审核时间：

## 节奏审核
| 检查项 | 状态 | 备注 |
|--------|------|------|
| 开场速度 | ✅/⚠️/❌ | |
| 情节点密度 | ✅/⚠️/❌ | |

## 冲突审核
| 检查项 | 状态 | 备注 |
|--------|------|------|
| 冲突强度 | ✅/⚠️/❌ | |
| 冲突升级 | ✅/⚠️/❌ | |

## 爽点审核
| 检查项 | 状态 | 备注 |
|--------|------|------|
| 爽点密度 | ✅/⚠️/❌ | |

## 悬念审核
| 检查项 | 状态 | 备注 |
|--------|------|------|
| 结尾钩子 | ✅/⚠️/❌ | |

## 问题清单
1. [问题] - 第 X 集
2. [问题] - 第 X 集

## 修改建议
1. [建议]
2. [建议]

## 结论
[通过/需修改/不通过]
```

## 常见问题

| 问题 | 表现 | 修改建议 |
|------|------|----------|
| 节奏太慢 | 开场超过 30 秒没冲突 | 删减铺垫，直接进冲突 |
| 冲突不够 | 对话太温和 | 加强语气，增加肢体冲突 |
| 爽点不足 | 主角太憋屈 | 增加反击，提前打脸 |
| 悬念不够 | 结尾太平 | 增加意外揭示/危险 |
| 台词太长 | 一句超过 30 字 | 拆分成短句 |
