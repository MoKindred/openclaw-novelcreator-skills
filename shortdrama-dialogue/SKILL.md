# shortdrama-dialogue - 短剧台词优化技能

## 功能描述
优化短剧台词，使其简短有力、有画面感、符合人设、推动剧情。

## 台词核心原则

### 1. 简短有力
> 台词超过两行必须精简。

### 2. 有画面感
### 3. 推动剧情
### 4. 符合人设

| 人设 | 台词风格 | 示例 |
|------|----------|------|
| 霸道总裁 | 简短、命令式 | "过来。""刷我的卡。" |
| 绿茶女配 | 柔弱、暗示 | "姐姐不会生气吧？" |
| 战神男主 | 冷峻、有威慑力 | "动她试试。" |
| 甜妹女主 | 活泼、可爱 | "哇！好厉害！" |

## 台词技巧库

1. **潜台词** - 说出来的话 vs 心里的话
2. **反问句** - 增强语气
3. **重复强调** - 强化情绪
4. **沉默的力量** - 不说比说更有力
5. **打断** - 制造紧张感

## 质量检查
- [ ] 每句台词都有目的吗？
- [ ] 台词超过两行了吗？
- [ ] 符合角色人设吗？
- [ ] 有画面感吗？
- [ ] 有水词吗？
