# 短剧剧本写作技能包

## 📦 技能列表

| 技能名 | 功能 | 使用场景 |
|--------|------|----------|
| `shortdrama-format` | 剧本格式规范化 | 将任意文本转为标准短剧格式 |
| `shortdrama-structure` | 结构设计与爽点规划 | 分析/设计黄金开局、爽点、钩子 |
| `shortdrama-character` | 人物设定与角色塑造 | 创建角色档案、关系图 |
| `shortdrama-outline` | 大纲与付费卡点规划 | 100 集分集大纲、卡点设计 |
| `shortdrama-dialogue` | 台词优化 | 让台词简短有力、有画面感 |
| `shortdrama-adapt` | 小说改编短剧 | 将小说/网文改编成短剧剧本 |

## 🚀 使用方式

### 方式一：直接调用技能
```
帮我用 shortdrama-format 技能把这个故事改成剧本...
```

### 方式二：自然语言请求
```
帮我写一个闪婚甜宠剧的第 1 集剧本
```

### 方式三：组合使用
```
1. 先用 shortdrama-character 创建人物设定
2. 再用 shortdrama-outline 规划 100 集大纲
3. 最后用 shortdrama-format 写具体剧本
```

## 💡 典型工作流

### 从零创作
```
1. 确定题材和核心创意
2. shortdrama-character → 创建人物设定
3. shortdrama-outline → 规划 100 集大纲和卡点
4. shortdrama-format → 逐集创作剧本
5. shortdrama-dialogue → 优化台词
6. shortdrama-structure → 检查爽点和钩子
```

### 小说改编
```
1. shortdrama-adapt → 提炼核心冲突
2. shortdrama-outline → 重新规划结构
3. shortdrama-format → 转换为剧本格式
4. shortdrama-dialogue → 优化台词
```

## 📊 短剧核心数据

| 指标 | 标准值 |
|------|--------|
| 单集时长 | 1-2 分钟 |
| 单集字数 | 500-800 字 |
| 总集数 | 80-100 集 |
| 爽点密度 | ≥3 个/集 |
| 角色数量 | ≤20 个/集 |
| 付费卡点 | 第 10/30/60 集 |
| 开局冲突 | 3 秒内 |

## 🎯 爆款公式

```
黄金开局（3 秒）→ 冲突升级（30 秒）→ 反转打脸（60 秒）→ 钩子结尾（90 秒）
```
