# Director Skill - 导演讲戏技能

## 核心理念

**导演不是复述剧本，而是把脑海中的画面完整描述出来。**

剧本写："她回到空荡荡的家，想起了过去"

导演讲戏：
> 她推开门，玄关的感应灯没亮，只有客厅窗帘缝透进来一道惨白的路灯光。她没开灯，摸黑走到沙发前坐下，目光落在茶几上那只没洗的杯子上。镜头从她的侧脸缓缓推向杯子的特写，冷白路灯光打在她半边脸上，另一半隐没在暗处。

---

## 讲戏规则

### 1. 用具体的物理动作，不用抽象概念

| ❌ 抽象 | ✅ 具体 |
|--------|--------|
| "她很悲伤" | "她眼眶红了，咬着下唇，眼泪在眼眶里打转但没掉下来" |
| "他很愤怒" | "他拳头攥紧，指节发白，脖子上的青筋暴起" |
| "她紧张" | "她手指不自觉地绞着衣角，呼吸变浅，眼神飘忽" |
| "空荡荡的家" | "玄关的感应灯没亮，客厅只有窗帘缝透进来的路灯光" |

### 2. 用连续的动作链，让画面"动起来"

| ❌ 静态 | ✅ 连续动作 |
|--------|------------|
| "她坐在沙发上" | "她推开门→没开灯→摸黑走到沙发前→缓缓坐下→目光落在茶几上" |
| "他站起来" | "他双手撑住桌子→缓缓起身→椅子在地板上划出刺耳的声音" |

### 3. 镜头运动要有明确方向

| 镜头 | 描述示例 |
|------|----------|
| **推镜** | "镜头从她的侧脸缓缓推向杯子的特写" |
| **拉镜** | "镜头从戒指特写缓缓拉远，露出她整个人坐在床边" |
| **摇镜** | "镜头从门口缓缓摇向窗边，扫过空荡的客厅" |
| **跟镜** | "镜头跟在她身后，保持半步距离，轻微晃动" |
| **固定** | "镜头固定在她脸上，捕捉每一个细微表情变化" |

### 4. 光影描述要具体

| 维度 | 描述示例 |
|------|----------|
| **方向** | "冷白路灯光从左侧窗帘缝射入，打在她半边脸上" |
| **色温** | "暖黄台灯光（3000K）与冷白月光（6000K）形成对比" |
| **强度** | "光线很弱，只能勉强勾勒出她的轮廓" |
| **对比** | "她半边脸在光里，另一半隐没在暗处" |

## 讲戏格式

### 完整讲戏模板

```markdown
## 场景 X-X [地点] [时间]

### 画面内容
[具体的物理环境描述，包含光影、道具、空间关系]

### 人物动作
[连续的动作链，从入场到结束，每个动作都有明确指向]

### 镜头运动
[镜头如何运动，从哪到哪，速度如何]

### 光影氛围
[光源方向、色温、强度、对比]

### 台词声音
[台词 + 语气 + 环境音]
```

### 融合叙述模板（推荐）

```markdown
## 场景 X-X [地点] [时间]

[自然流畅的叙述，融合所有维度]

> 她推开门，玄关的感应灯没亮，只有客厅窗帘缝透进来一道惨白的路灯光。她没开灯，摸黑走到沙发前坐下，目光落在茶几上那只没洗的杯子上。镜头从她的侧脸缓缓推向杯子的特写，冷白路灯光打在她半边脸上，另一半隐没在暗处。远处传来楼下汽车的鸣笛声，很闷，像隔着一层水。
```

## 讲戏流程

### 步骤 1：读取剧本

读取 `01-short-drama-script.md` 中的场景描述

### 步骤 2：视觉化翻译

将抽象描述转化为具体画面：

| 剧本 | 导演讲戏 |
|------|----------|
| "叶辰隐忍" | "叶辰低头，喉结滚动了一下，拳头攥紧又松开，指甲掐进掌心" |
| "王桂芳不屑" | "王桂芳嘴角向下撇，眼睛从鼻梁上方看人，鼻孔里哼出一声冷笑" |
| "苏清雪委屈" | "苏清雪眼眶瞬间红了，咬着下唇，眼泪在眼眶里打转，声音发颤" |

### 步骤 3：设计动作链

为每个场景设计连续动作：

```
入场→位置→动作→反应→退场
```

### 步骤 4：设计镜头

为每个动作链设计镜头运动：

```
建立镜头→主体镜头→细节镜头→反应镜头
```

### 步骤 5：设计光影

为每个场景设计光影氛围：

```
主光源方向 + 色温 + 强度 + 辅助光 + 对比度
```

## 讲戏示例

### 示例 1：龙王归位

**剧本原文**：
> 将士统领单膝跪地，高声："末将参见龙王殿主！"

**导演讲戏**：
> 黑衣将士统领大步走到叶辰面前三米处，突然停住，右腿向后撤半步，左腿弯曲，单膝跪地，动作干脆利落。镜头从叶辰的侧脸快速摇向跪地的将士，再拉远到全景，露出身后数百名整齐列队的黑衣保镖。将士统领抬头，声音洪亮穿透清晨的空气："末将参见龙王殿主！"声音落下的瞬间，身后所有保镖齐刷刷单膝跪地，铠甲摩擦声整齐划一。冷白晨光从东侧射入，打在叶辰身上，形成一道金色轮廓光。

### 示例 2：离婚逼迫

**剧本原文**：
> 王桂芳把离婚协议甩在叶辰脸上

**导演讲戏**：
> 王桂芳从茶几上抓起离婚协议，手腕向后扬起，手肘发力，纸张在空中划出一道弧线，"啪"地一声甩在叶辰脸上。镜头固定在中景，捕捉纸张飘落的慢动作，最后落在叶辰脚边。叶辰没动，目光从飘落的纸张移向王桂芳，眼神里有什么东西碎了。暖黄客厅灯光下，王桂芳的脸因愤怒而扭曲，叶辰的脸隐没在阴影里。

## 注意事项

1. **不讲抽象**：永远不用"悲伤"、"愤怒"等抽象词
2. **动作连续**：每个动作都有前因后果
3. **镜头明确**：镜头运动有起点、终点、速度
4. **光影具体**：方向、色温、强度都要明确
5. **融合叙述**：不用条列式，用自然流畅的叙述
