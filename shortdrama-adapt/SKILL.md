# shortdrama-adapt - 小说改编短剧剧本技能

## 功能描述
将小说/网文改编成短剧剧本，提炼核心冲突，转换为短剧格式和节奏。

## 改编流程

```
小说原文 → 核心提炼 → 结构重组 → 格式转换 → 节奏优化
```

## 核心提炼

### 1. 提取核心冲突
- 主角的最大困境
- 主要矛盾关系
- 关键转折点

### 2. 角色精简
- 小说：可能有几十个人物
- 短剧：单集有台词的角色≤20 个

### 3. 情节压缩
- 删除次要情节和冗长叙述
- 突出关键冲突和情感高潮
- 将内心独白转化为对话或肢体语言

## 结构重组

### 小说结构 vs 短剧结构

| 维度 | 小说 | 短剧 |
|------|------|------|
| 开篇 | 可以慢热铺垫 | 3 秒必须冲突 |
| 节奏 | 张弛有度 | 持续高密度 |
| 爽点 | 偶尔出现 | 每集 3-5 个 |
| 结尾 | 可以平淡过渡 | 必须留钩子 |

## 改编检查清单

- [ ] 核心冲突提取准确吗？
- [ ] 角色数量精简了吗（≤20 个/集）？
- [ ] 情节压缩合理吗？
- [ ] 格式转换正确吗？
- [ ] 节奏够快吗（3 秒冲突）？
- [ ] 爽点够密吗（≥3 个/集）？
- [ ] 结尾有钩子吗？
