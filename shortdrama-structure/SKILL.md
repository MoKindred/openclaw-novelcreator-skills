# shortdrama-structure - 短剧结构设计与爽点规划技能

## 功能描述
分析或设计短剧的结构，包括黄金开局、爽点分布、反转设计、钩子设置等。

## 核心公式

### 时间结构
```
0-3 秒：冲突锚定（抓住注意力）
3-30 秒：情绪爆破（激发情绪）
30-90 秒：反转打脸（制造爽感）
90-120 秒：悬念钩子（引导下一集）
```

### 事件公式
```
事件 → 反应 → 反转 → 再反应
```

## 爽点类型库

### 1. 打脸爽
- 被看不起 → 展现实力 → 对方震惊
- 示例：赘婿亮出首富身份

### 2. 逆转爽
- 绝境 → 转机 → 反杀 → 碾压
- 示例：公司破产→发现隐藏资产

### 3. 升级爽
- 弱小 → 获得能力 → 变强
- 示例：获得系统/觉醒异能

### 4. 恋爱爽
- 高冷 → 动心 → 宠溺 → 独占
- 示例：总裁独宠小娇妻

## 钩子类型库

### 1. 悬念钩
- "门打开，门外的人竟然是..."
- "她打开信封，脸色煞白"

### 2. 危机钩
- "再不给钱，明天就收房！"
- "手术费还差 50 万"

### 3. 情感钩
- "如果再来一次，你还会选择他吗？"
- "三年后，他们再次相遇"

### 4. 反转钩
- "你以为这就结束了？"
- "真正的幕后黑手，竟然是他！"

## 100 集爽点分布

| 集数区间 | 爽点密度 | 重点类型 |
|----------|----------|----------|
| 1-10 集 | 每集 3-5 个 | 身份悬念、初遇爽 |
| 11-30 集 | 每集 4-6 个 | 打脸、护妻、宠溺 |
| 31-60 集 | 每集 4-6 个 | 复仇、逆袭、情敌对战 |
| 61-90 集 | 每集 5-8 个 | 终极对决、身份曝光 |
| 91-100 集 | 每集 3-5 个 | 圆满、温情、收尾 |

## 质量检查
- [ ] 前 3 秒有冲突吗？
- [ ] 30 秒内有情绪爆破吗？
- [ ] 每集有反转吗？
- [ ] 结尾有钩子吗？
- [ ] 爽点密度够吗（≥3 个/集）？
- [ ] 反转合理有铺垫吗？
